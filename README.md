# ⚡ NEON SYNDICATE: THE HEIST

An advanced text-based "Choose Your Own Adventure" game built for the **GitHub Copilot Challenge**. This project explores AI-assisted development, state management in Python, and terminal-based UI design.

## 🚀 Overview
In **Neon Syndicate**, you play as a covert operative infiltrating the high-security Megacorp Tower. To succeed, you must navigate through restricted areas, manage your inventory, and outsmart a Sentinel AI.

## 🛠️ Built With Copilot
This application was developed using **GitHub Copilot**. The AI assisted in:
* **Terminal Effects:** Generating the `slow_print` function to create a "hacker" typing aesthetic.
* **Game Logic:** Designing a modular room-based system and an inventory dictionary to track the Level 4 Keycard.
* **Visuals:** Creating the ASCII art boot sequence and "Megacorp" branding.
* **Error Handling:** Implementing input validation to ensure the game doesn't crash on invalid commands.
